# BMI Calculator

A simple and interactive BMI Calculator built using HTML, CSS, and JavaScript.

## Features
- Calculates Body Mass Index (BMI)
- Displays BMI category:
  - Underweight
  - Normal
  - Overweight
- Introduce a pre-defined JS library for confetti
- Confetti celebration for Normal BMI
- Recalculate / Reset functionality
- Clean and responsive UI

## Tech Used
- HTML
- CSS
- JavaScript

## How to Use
1. Enter height in meters
2. Enter weight in kilograms
3. Click **Calculate**
4. View your BMI result
